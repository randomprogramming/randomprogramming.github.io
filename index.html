HTML

HTML je standardni jezik označavanja za dokumente dizajnirane za prikaz u web pregledniku. Uz njega koristimo jezik stiliranja poput CSS-a i jezik skriptiranja kao što je JavaScript.

Web preglednici primaju HTML dokumente s web poslužitelja ili iz lokalne pohrane i prikazuju ispis dokumenta na multimedijske web stranice. HTML opisuje strukturu web stranice semantički i izvorno sadržava naznake za izgled dokumenta.

HTML elementi su sastavni dijelovi HTML stranica. S HTML konstrukcijama, slikama i drugim objektima, poput interaktivnih oblika, mogu biti ugrađeni u prikazanu stranicu. HTML pruža sredstvo za stvaranje strukturiranih dokumenata označavanjem strukturne semantike teksta kao što su: naslovi, odlomci, popisi, poveznice, citati i druge oznake. HTML elementi su razgraničeni oznakama, napisane uglastim zagradama. Oznake kao što su <img /> i <input /> izravno uvežu sadržaj u stranicu. Ostale oznake kao što <p > okružuju i pružaju informacije o tekstu dokumenta, a može sadržavati i druge oznake kao pod-elemente. Preglednici ne prikazuju HTML oznake, ali koristite ih za tumačenje sadržaja stranice.

HTML 5.0

Objavljeno: 28. listopad 2014.

Ova specifikacija definira 5. veliku reviziju glavnog jezika na World Wide Webu: Hypertext Markup Language-a. U ovoj verziji uvode se nove značajke za pomoć autorima web aplikacija, uvode se novi elementi temeljeni na istraživanju prevladavajuće autorske prakse, a posebna pažnja posvećena je definiranju jasnih kriterija usklađenosti za korisničke agente u nastojanju da se poboljša interoperabilnost.

Oznake poput <section />, <article />, <footer /> pomažu nam bolje rasporediti elemente unutar HTML dokumenta te svakom elementu zadati njegovu svrhu, kako bi znali točno čemu svaki element služi. sa značenjem:
  1. <header> - predstavlja skup uvodnih ili navigacijskih pomagala, u većini slučaja bi sadržavao logotip ili ime tvrtke
  2. <nav> - predstavlja odjeljak dokumenta namijenjen navigaciji
  3. <section> - predstavlja generički dokument ili odjeljak web stranice. Koristimo ga zajedno s elementima <h1 />, <h2 />, <h3 />,<h4 />, <h5 /> i <h6 /> za označavanje strukture dokumenta
  4. <article> - predstavlja neovisan dio dokumenta dokumenta, poput bloga ili članaka u novinama
  5. <aside> - predstavlja dio sadržaja koji je tek neznatno povezan s ostatkom stranice, primjer bi bio navigacija unutar jedne web stranice
  6. <footer> - predstavlja podnožje za odjeljak i može sadržavati podatke o autoru, autorskim pravima i ostale informacije

Prijašnje spomenut <input /> element, koji služi za korisnički unos datuma, e-maila i drugih informacija , je dobio nekoliko novih tipova. Oni se koriste na sljedeči način: <input type="tip_unosa" />

    Novi tipovi unosa
  1. tel - definira polje za unos telefonskog broja
  2. search - definira polje u koje se unosi vrijednost koju korisnik želi pretražiti
  3. url - definira polje koje prima samo URL
  4. email - definira polje koje prima samo e-mail adresu
  5. date - definira polje u kojem korisnik može odabrati neki datum
  6. time - definira polje u kojem korisnik može odabrati neko vrijeme
  7. number - definira polje koje prima samo neki broj kao svoju vrijednost
  8. range - definira polje za odabir neke vrijednosti u kojem točna vrijednost nije toliko važna, npr. filtar cijene na web shopu
  9. color - definira polje koje prima neku boju kao svoju vrijednost

HTML5 je također uveo neke nove atribute. Atributi su svojstva koja modificiraju neki element, na primjer, gore spomenut <input type="tip_unosa" /> ima atribut type uz mnoge druge. Među najvažnijim novim atributima je required, koji se može stavit uz element <input />. On zahtijeva da neki <input /> element mora imati vrijednost prije nego što se forma preda.

Pošto elementi u ovoj formi imaju required atribut, forma se ne može predat ukoliko je neko polje prazno. Gore navedeni elementi također sadrže još jedan novi atribut, placeholder. placeholder je namijenjen kako bi korisniku rekao što se treba nalaziti unutar tog elementa.

Navedeni element ima placeholder="Ime" te on izgleda ovako: <input required type="text" placeholder="Ime"/>

HTML 5.1

Objavljeno: 1. studeni 2016.

This specification defines the 5th major version, first minor revision of the core language of the World Wide Web: the HTML. In this version, new features continue to be introduced to help Web application authors, new elements continue to be introduced based on research into prevailing authoring practices, and special attention continues to be given to defining clear conformance criteria for user agents in an effort to improve interoperability.

Objavljeno: 14. prosinac 2017.

In this version, new features continue to be introduced to help Web application authors, new elements continue to be introduced based on research into prevailing authoring practices, and special attention continues to be given to defining clear conformance criteria for user agents in an effort to improve interoperability.
